<p>A dotnet 6 E-commerce API built with following features :</p>
<ul>
    <li>JWT token authentication</li>
    <li>Global error log in separate file using Log4Net</li>
    <li>Global Trace log in separate file using Log4Net</li>
    <li>Swagger</li>
    <li>AutoMapper</li>
    <li>Role based authorization</li>
    <li>Identity</li>
    <li>Accounts Controller</li>
    <li>Products Controller</li>
    <li>Extention Method</li>
    <li>Dynamic Pagination</li>
    <li>Custom Response</li>
    <li>Entity framework core</li>
    <li>Unit testing</li>
    <li>DB Schema Project</li>
</ul>
